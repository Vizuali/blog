_Let's talk about grayscale images and superfast processing._

Well... This is something different. This is a implementation with Shaders in Processing of Gray Scale images.
Here is the repo if you want to see the code behind this.
https://github.com/Vizuali/convolutional-shaders

![Gif](https://github.com/Vizuali/convolutional-shaders/blob/master/assets/GrayScale.gif?raw=true)

In this repo we implemented all convolution and gray scale filters, here is a little guide to use it:

- __Box Blur:__ Press 'v'
- __Edge 1:__ Press 'e'
- __Edge 2:__ Press 'r'
- __Edge 3:__ Press 't'
- __Sharp:__ Press 's'
- __BloxBlur:__ Press 'v'
- __GrayScale Average RGB:__ Press 'g'
- __GrayScale Luma601:__ Press 'h'
- __GrayScale Luma709:__ Press 'j'
- __GrayScale Luma240:__ Press 'k'
- __GrayScale Luma2020:__ Press 'l'
- __Identity:__ Press 'i' (This is useless but maybe someone want to see the same image twice)
